## 安装nginx
docker run -d --restart unless-stopped --privileged=true -p 80:80 --name nginx nginx
apt update
apt install -y vim

#### 配置
- 修改 iptv.conf
  - cd /etc/nginx/conf.d/
  - vim iptv.conf

```bash
server {
    listen       80;
    listen  [::]:80;
    server_name  iptv.home;

    location / {
        proxy_pass   http://192.168.100.1:8081;
        proxy_set_header Host $http_host;
    }
}

```
  - vim router.conf
  - 
